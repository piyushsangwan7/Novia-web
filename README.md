# 🎬 Novia

Novia is a modern movie streaming platform UI built using **HTML5** and **CSS3**. It features a clean and visually appealing interface inspired by modern entertainment platforms.

## 🚀 Technologies Used

- HTML5
- CSS3

## ✨ Features

- Modern UI
- Hero section
- Movie & TV show cards
- Hover effects
- Clean layout
